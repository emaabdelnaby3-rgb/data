# المنصة الحكومية الموحدة للجمعيات الخيرية (MVP)

منصة موحدة تربط 3 جمعيات خيرية في نظام واحد لمنع تكرار صرف المساعدات لنفس المحتاج، مع إدارة حالات وتبرعات وداشبورد.

## أهم ما تم تنفيذه
- واجهة عربية كاملة مع **قائمة جانبية** بدل الشريط العلوي.
- تسجيل متبرع (بالإيميل + الرقم القومي + كلمة مرور).
- تسجيل دخول بالرقم القومي/الإيميل (المحتاج يعتمد على الرقم القومي + كلمة المرور).
- استمارة المحتاج تشمل:
  - السن
  - عدد الأطفال
  - الدخل
  - متزوج أم لا
  - لديه عمل أم لا
  - الراتب
  - وصف الحالة
  - رفع مستندات: صورة البطاقة + شهادات الميلاد + مستندات إضافية
- قاعدة بيانات تشمل جمعيات + حالات تجريبية مرتبطة بكل جمعية.
- التبرع بطرق دفع: **فودافون كاش / انستا باي / فيزا**.
- منطق توزيع التبرع:
  - المطلوب للحالة يذهب للمحتاج.
  - الزيادة تذهب لحساب الجمعية.
- إيصال لكل عملية تبرع (`receipt_code`) + داشبورد تفاعلي.

## التشغيل
```bash
cd backend
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload
```

ثم افتح:
- Home: `http://127.0.0.1:8000/`
- API docs: `http://127.0.0.1:8000/docs`
